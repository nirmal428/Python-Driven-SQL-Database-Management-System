# Python-Driven UI for Advanced SQL Database Operations

This project is a Python-driven web application that provides a user-friendly interface for advanced SQL database operations. 
Built using Streamlit and MySQL, it allows users to view, manage, and analyze database records without writing any SQL queries.

It is particularly useful for managers, team leads, and non-technical users who need to interact with database systems in a business environment.

---

## 🛠️ Technical Stack
* **Frontend:** Python (Streamlit)
* **Backend:** MySQL
* **Data Handling:** Pandas & NumPy
* **Database Logic:** Stored Procedures, Functions, and Views
